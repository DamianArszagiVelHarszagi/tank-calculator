# TankCalc

A fuel cost calculator for road trips. Add stops on a map, set your car details, and see how much you will spend on fuel — broken down by country, with refuel suggestions along the way.

**Live:** [damianarszagivelharszagi.github.io/tank-calculator](https://damianarszagivelharszagi.github.io/tank-calculator)

---

## What it does

- Calculates total fuel cost for any route
- Shows fuel prices per country you drive through
- Suggests where to refuel based on your tank range and cheaper countries nearby
- Splits the cost between multiple people
- Supports return trips

Prices are looked up in the background per stop and update automatically once fetched.

---

## How to use

1. Type a city in the search bar and press Enter
2. Add as many stops as you need
3. Set your fuel type, consumption (L/100km), and tank size
4. Set the number of people and check return trip if needed
5. Click Calculate
